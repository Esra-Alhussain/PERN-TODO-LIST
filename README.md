# PERN Todo List

![PERN Todo List Preview]

## Overview

The PERN Todo List is a full-stack web application built using the PERN (PostgreSQL, Express, React, Node.js) stack. It allows users to create, view, edit, and delete todo items. This project serves as a practical demonstration of integrating various technologies to develop a functional web application.

## Features

- **Todo Management:** Users can create, view, edit, and delete todo items.
- **Responsive Design:** The application is designed to be responsive and accessible on different devices.

## Technologies Used

- **Frontend:**
  - React.js
  - HTML
  - CSS
  - JavaScript

- **Backend:**
  - Node.js
  - Express.js
  - PostgreSQL
